# MoreBeauty

一个纯静态的图片画廊页面：

- 首页展示图集封面瀑布流
- 点击封面后，在弹层中查看该图集的全部图片
- 数据来源于本地 `vipPicture.json`

## 本地运行

这个项目使用 `fetch('vipPicture.json')` 读取数据，所以不要直接双击 `index.html` 用 `file://` 打开。

推荐在项目目录启动一个本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/
```

## 文件说明

- `index.html`：页面结构和基础样式
- `index.js`：瀑布流布局、弹层交互、数据加载
- `vipPicture.json`：图集数据
